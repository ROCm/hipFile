# hipFile Code of Conduct

hipFile is a part of ROCm and, like other ROCm components,
requires that engagement with the hipFile community abides by the
[GitHub community guidelines](https://docs.github.com/en/site-policy/github-terms/github-community-guidelines)
and the
[GitHub community code of conduct](https://docs.github.com/en/site-policy/github-terms/github-community-code-of-conduct).
